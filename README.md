### Control USV
- Hardware: OrangePi PC plus.
- OS: ARMBIAN (base Debian).
- Sensors and others: adafruit_ina219, adafruit_pca9658, hmc5883l, mpu6050.
- GPS: Adrusimple RTK F9P
